# KMA Archive Experience

KMA Archive Experience is a modern academic publishing platform designed to manage the complete research lifecycle — from manuscript submission to peer review and final publication.

The system features a role-based architecture with dedicated dashboards for Authors, Reviewers, and Administrators, along with a public landing page for research discovery and a secure pay-per-article access model.

---

## ✨ Key Features

- 📄 **Author Portal**
  - Submit and manage research papers
  - Track review progress and revisions

- 🧠 **Reviewer Portal**
  - Evaluate assigned manuscripts
  - Provide structured academic feedback

- 🏛️ **Admin Dashboard**
  - Full control over article lifecycle
  - Reviewer verification and publication management

- 🌐 **User Landing Page**
  - Browse and search research articles
  - View abstracts with paywalled full access

- 💳 **Secure Access Model**
  - Login-based article access
  - Pay-per-article unlocking system
